# Contenido 

---

Desarrolla un ETL que recopile datos históricos y en tiempo real de múltiples criptomonedas. Crea un dashboard interactivo con análisis técnico y construye un modelo predictivo para variaciones de precios.

**API:** https://dogapi.dog/docs/api-v2

##### Entregables

1. **Extractor ETL:** Script que obtiene datos de 10+ criptomonedas cada 15 minutos
2. **BD PostgreSQL:** Tablas para criptomonedas, precios históricos y mercados
3. **Dashboard Streamlit:** Gráficos de precios, comparativas 24h, análisis de volatilidad
4. **Docker Compose:** Contenedores para base de datos, ETL y dashboard
5. **ML Jupyter:** Modelo ARIMA o Prophet para predicción de precios futuros
6. **Presentación PPT:** Problema identificado, alcance del proyecto, pregunta de investigación, herramientas usadas, arquitectura de datos, modelos implementados, métricas de evaluación, resultados obtenidos y recomendaciones

#### 💡 Sugerencias de Análisis

- Calcular indicadores técnicos (RSI, MACD, Bandas de Bollinger)
- Detectar patrones de negociación
- Analizar correlación entre diferentes criptomonedas
- Predecir cambios de precio basados en volumen
